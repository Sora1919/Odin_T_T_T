# Odin_T_T_T

Features of this Tic Tac Toe Game:

1.Complete Game Functionality:
  - Players take turns placing X's and O's
  - Win detection for rows, columns, and diagonals
  - Draw detection when the board is full

2.Visual Design:
  - Modern, colorful interface with a dark theme
  - Responsive layout that works on mobile and desktop
  - Visual feedback for player turns and winning cells
  - Smooth hover effects and transitions

3. Game State Tracking:
  - Score tracking for both players and draws
  - Clear indication of whose turn it is
  - Win/draw message display

4.Controls:
  - Restart button to reset the current game
  - Reset scores button to clear all scores
  - Instructions:
  - Clear instructions for how to play the game
